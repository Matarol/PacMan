# 🎮 Pac-Man (JavaScript)

A browser-based Pac-Man clone built with **vanilla JavaScript**, HTML, and CSS.

## 🚀 Live Demo

👉 https://matarol.github.io/PacMan/

## 📸 Preview

![Gameplay](./assets/screenshot.png)

## 🕹️ Controls

* ⬅️ ➡️ ⬆️ ⬇️ Arrow keys to move

## ✨ Features

* Classic Pac-Man movement
* Ghost enemies
* Collision detection
* Score tracking
* Game over conditions

## 🛠️ Built With

* HTML5
* CSS3
* Vanilla JavaScript (no frameworks)

## ▶️ Run Locally

1. Clone the repo:

   ```bash
   git clone https://github.com/Matarol/PacMan.git
   ```
2. Open `index.html` in your browser
